Sublime Text插件推荐

Posted on 2018-09-20 | In 清单 | |

插件安装

  • Package Control

    功能：安装插件
    快捷键：Ctrl + Shift + P，输入 Install 选择 Install Package

中文支持

  • ChineseLocalizations

    功能：汉化
    菜单：Help -> Language

  • ConvertToUTF8

    功能：将文件编码从GBK转换成UTF8
    菜单：File -> Set File Encoding to / Reload with Encoding

  • codecs33

    功能：ConvertToUTF8在sublime3中无法使用时安装

主题

  • Material Theme

    功能：Material主题

  • A File Icon

    功能：显示文件图标，Material主题需要安装

  • ColorSublime

    功能：主题预览安装

  • MarkdownExtended

    功能：markdown显示主题

界面

  • Side​Bar​Enhancement

    功能：侧边栏增强工具，文件定义引用跳转

  • BracketHighlighter

    功能：括号、引号匹配高亮显示

格式化

  • Alignment

    功能：进行智能对齐，选中代码
    快捷键：ctrl+alt+a

  • AutoPep8

    功能：格式化Python代码
    快捷键：ctrl+8（预览），ctrl+shift+8（格式化）

  • CodeFormatter

    功能：自动格式化代码，支持多种编程语言，需要指定编程语言路径（html格式化不好用，内联的script标签会一直缩进）
    快捷键：ctrl+alt+f

  • HTML-CSS-JS Prettify

    功能：格式化html，css，js代码（好用）
    快捷键：command+shift+h

编程

  • Emmet

    功能：前端开发神器，缩写组合快速编写html

  • Jedi

    功能：Python智能代码补全

  • DocBlockr

    功能：自动补全注释插件

  • SublimeLinter

    功能：语法检查工具，针对不同编程语言还需要单独安装插件，如linter-php

  • SublimeCodeIntel

    功能：代码智能提示

  • HTML5

    功能：HTML snippets(可选)

  • JQuery

    功能：JQuery snippets

Markdown

  • Markdown Editing

    功能：高亮显示Markdown语法

  • OmniMarkupPreviewer

    功能：预览markdown 编辑的效果，同样支持渲染代码高亮的样式
    快捷键：super+alt+o（网页预览），super+alt+x（导出）

辅助

  • FileDiffs

    功能：对比文件区别

  • SublimeTmpl

    功能：提供常用文件的模板

  • SublimeREPL

    功能：多种脚本语言的执行环境，可以方便的进入代码所在目录的shell或python环境

  • SFTP

    功能：FTP工具

  • GIT

    功能：GIT工具
