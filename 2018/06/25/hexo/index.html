###github上搭建Hexo博客

https://www.cnblogs.com/visugar/p/6821777.html

  1. 安装git for windows

    • 下载地址

    • 安装：一直点击下一步

    • 查看版本：

git version

  1. 安装NodeJs

    • 下载地址

    • 安装：一直点击下一步，Custom Setup中勾选 Add to PATH

    • 查看版本：

node -v

  1. 安装hexo

    • 创建blog目录，进入目录
        mkdir blogdir

        cd blogdir

- 安装命令：`npm -i -g hexo`

- 查看版本：



        hexo -v
  1. 构建hexo项目，本地测试
- 生成hexo代码



        hexo init

        hexo clean

        hexo generate

- 启动本地服务器



        #注：hexo 3.0把服务器独立成个别模块，需要单独安装

        npm install hexo-server --save

        hexo server

- 打开浏览器测试： `http://localhost:4000`
  1. gitbash中配置git环境，生成ssh密钥
- 执行



        git config --global user.name "YourName"

        git config --global user.email "YourEmail"



- 创建ssh公钥



        ssh-keygen -t rsa -C "yourmail@example.com"

    生成ssh，找到id_rsa.pub文件内容
  1. 申请github账号，创建repo，配置免密登录
- 申请账号，[github地址](https://github.com/)



- 创建仓库： 

    - 新建仓库

    - 仓库名：yourname.github.io

- 配置免密登录：

    - Setting——SSH and GPG keys——New SSH key——粘贴刚创建id_rsa.pub公钥文件内容

    - 验证是否成功 `ssh -T git@github.com`

  1. 部署hexo

    • 安装部署工具
        npm install hexo-deployer-git --save

- 配置部署参数



    编辑_config.yml，最下方添加内容



        deploy:

          type: git

          repo: git@github.com:yourname/yourname.github.io.git

          branch: master



    repo也可以是https://github.com/yourname/yourname.github.io.git

 - 上传到github

     执行命令

         hexo clean
         hexo generate
         hexo deploy

  1. 修改配置内容

    • 选择主题
    下载主题[下载地址](https://hexo.io/themes/)，如`next`[下载地址](https://github.com/iissnan/hexo-theme-next)，解压到themes目录下，配置 `themes/next/_config.yml`



- 导航栏



- 侧边栏

  1. hexo使用

    • 发布新文章

      在source->_posts目录下，存放要发布的文章
